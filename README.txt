Welcome to PinkFloydAlbums app.
In the app you will find the collection of all band albums released,
with details, album cover picture and the user personal rank (1-10) for every album in the collection.

To run the app, the user will need to connect to Expo service, explained here:
https://expo.io/
(Download as zip, type in command line:
    npm install
    expo start
)

The app was developed and test on Android emulator, on google pixel device:
Google pixel 2, 1440 X 2880, 560dpi, Android 8.0
