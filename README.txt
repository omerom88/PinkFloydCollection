Welcome to PinkFloydAlbum app.
In the app you will find the collection of all band albums released,
with details, picture and the user personal rank for every album in the collection.

To run the app, the user will need to connect to Expo service, explained here:
https://expo.io/

The app was developed and test on Android emulator, on devices:
Google pixel 2, 1440 X 2880, 560dpi
Galaxy S5 mini, 720 X 1280, xhdpi
